## Voltaとは

- Voltaは、Node.jsのバージョン管理ツールです。
- 開発環境ごとに異なるNode.jsやパッケージマネージャーの
  バージョン差異を防ぐために使用します。

## なぜVoltaを使用しているか

### 1. Node.jsのバージョンを統一するため

- プロジェクトごとにNode.jsのバージョンを固定できます。
  これにより、開発者間の環境差異を防止できます。

### 2. 自動的にバージョンが切り替わる

- プロジェクトディレクトリに移動すると、
  package.json に記載された設定に基づいて
  自動的に適切なNode.jsバージョンが使用されます。

### 3. グローバル環境を汚さない

- 従来の nvm などと異なり、
  シェルの再読み込みを必要とせず、
  軽量かつ高速に動作します。

## Volta のインストール

以下のコマンドを実行してください。

```bash
curl https://get.volta.sh/ | bash
```

### 設定の反映

インストール後、以下を実行して設定を反映します。

```
source ~/.profile
```

### Node.js（LTS）のインストール

```
volta install node@lts
```

## pnpm のインストール

本プロジェクトではパッケージマネージャーとして pnpm を使用します。

### 1. Volta 経由で pnpm をインストール

以下のコマンドを実行します。

```
volta install pnpm
```

### 2. インストール確認

```
pnpm -v
```

バージョンが表示されればインストール完了です。

### なぜ pnpm を使用するか

- npm よりも高速
- ディスク容量を節約できる（パッケージの実体をグローバルストアに保存し、
  各プロジェクトではハードリンクで参照する仕組み）
- モノレポ構成との相性が良い
- lockfile による依存関係の再現性が高い
